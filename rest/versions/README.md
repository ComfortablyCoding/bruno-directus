# versions

CRUD operations for Directus Content Versions. Supports individual and bulk operations as well as compare, save, and promote actions.

## Operations

- Query all versions
- Query a single version by ID
- Create a single version
- Create multiple versions (bulk)
- Update a single version
- Update multiple versions (bulk)
- Delete a single version
- Delete multiple versions (bulk)
- Compare a version against the current item
- Save changes to a version
- Promote a version to the main item
