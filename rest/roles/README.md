# roles

CRUD operations for Directus roles. Roles group users and assign policies for access control.

## Operations

- Query all roles
- Query a single role by ID
- Create a single role
- Create multiple roles (bulk)
- Update a single role
- Update multiple roles (bulk)
- Delete a single role
- Delete multiple roles (bulk)
