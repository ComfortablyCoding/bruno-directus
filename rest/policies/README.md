# policies

CRUD operations for Directus policies. Policies are new in Directus 11, used for granular access control — they define what actions are allowed on which collections and are assigned to roles.

## Operations

- Query all policies
- Query a single policy by ID
- Create a single policy
- Create multiple policies (bulk)
- Update a single policy
- Update multiple policies (bulk)
- Delete a single policy
- Delete multiple policies (bulk)
