# presets

CRUD operations for Directus presets. Presets store saved collection view configurations (layout, filters, search) for users.

## Operations

- Query all presets
- Query a single preset by ID
- Create a single preset
- Create multiple presets (bulk)
- Update a single preset
- Update multiple presets (bulk)
- Delete a single preset
- Delete multiple presets (bulk)
