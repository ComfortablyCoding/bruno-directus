# files

Manage files stored in Directus. Supports upload, import from URL, metadata updates, and deletion.

## Operations

- Query all files
- Query a single file by ID
- Upload a file (multipart form)
- Import a file from a URL
- Update file metadata
- Update multiple files (bulk)
- Delete a single file
- Delete multiple files (bulk)
