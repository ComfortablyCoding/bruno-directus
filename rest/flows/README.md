# flows

CRUD operations for Directus Flows. Supports individual and bulk operations as well as manual trigger endpoints.

## Operations

- Query all flows
- Query a single flow by ID
- Create a single flow
- Create multiple flows (bulk)
- Update a single flow
- Update multiple flows (bulk)
- Delete a single flow
- Delete multiple flows (bulk)
- Trigger a flow via GET
- Trigger a flow via POST
