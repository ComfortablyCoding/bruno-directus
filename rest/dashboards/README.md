# dashboards

CRUD operations for Directus Insights Dashboards. Supports individual and bulk operations.

## Operations

- Query all dashboards
- Query a single dashboard by ID
- Create a single dashboard
- Create multiple dashboards (bulk)
- Update a single dashboard
- Update multiple dashboards (bulk)
- Delete a single dashboard
- Delete multiple dashboards (bulk)
