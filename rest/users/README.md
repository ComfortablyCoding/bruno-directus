# users

Manage Directus users. Includes standard CRUD, bulk operations, current user endpoints, and invite flows.

## Operations

- Query all users
- Query a single user by ID
- Get the current authenticated user
- Create a user
- Create multiple users (bulk)
- Update a user
- Update the current authenticated user
- Update multiple users (bulk)
- Delete a user
- Delete multiple users (bulk)
- Invite a user by email
- Accept a user invite
