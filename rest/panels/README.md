# panels

CRUD operations for Directus Insights Dashboard Panels. Supports individual and bulk operations.

## Operations

- Query all panels
- Query a single panel by ID
- Create a single panel
- Create multiple panels (bulk)
- Update a single panel
- Update multiple panels (bulk)
- Delete a single panel
- Delete multiple panels (bulk)
