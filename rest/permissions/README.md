# permissions

CRUD operations for Directus permissions. Permissions define what actions a policy can perform on a given collection.

## Operations

- Query all permissions
- Query a single permission by ID
- Get current user's permissions
- Create a single permission
- Create multiple permissions (bulk)
- Update a single permission
- Update multiple permissions (bulk)
- Delete a single permission
- Delete multiple permissions (bulk)
