# fields

Manage fields (columns) within Directus collections. Supports querying across all collections or scoped to a specific collection.

## Operations

- Query all fields across all collections
- Query all fields in a specific collection
- Query a single field by collection and field name
- Create a field in a collection
- Update a field's metadata or schema
- Delete a field from a collection
