# shares

CRUD operations for Directus Shares. Supports individual and bulk operations as well as share info and authentication.

## Operations

- Query all shares
- Query a single share by ID
- Create a single share
- Create multiple shares (bulk)
- Update a single share
- Update multiple shares (bulk)
- Delete a single share
- Delete multiple shares (bulk)
- Get share info (public)
- Authenticate with a share
