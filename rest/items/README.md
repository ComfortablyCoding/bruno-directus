# items

CRUD operations for Directus collection items. Supports individual and bulk operations as well as singleton collections.

## Operations

- Query all items in a collection
- Query a single item by ID
- Create a single item
- Create multiple items (bulk)
- Update a single item
- Update multiple items (bulk)
- Delete a single item
- Delete multiple items (bulk)
- Get singleton item
- Update singleton item
