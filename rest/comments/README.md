# comments

CRUD operations for Directus comments. Comments are a dedicated collection in Directus 11 (previously stored within activity) and allow users to annotate items in any collection.

## Operations

- Query all comments
- Query a single comment by ID
- Create a single comment
- Create multiple comments (bulk)
- Update a single comment
- Update multiple comments (bulk)
- Delete a single comment
- Delete multiple comments (bulk)
